[ secrets ]  

  • to get the super wrench type "iseegold" while focused on the game at the map select scene
  • choose a map and the super wrench will be in your inventory select menu under the normal wrench
  • it flings all entities (bots, vehicles, bodies, bags) and turns them gold with a money effect on contact
  • does not save if removed from inventory and game is reloaded
  • you can also type "iseegold" while in a game to turn on a debug mode of some sort which displays entity information
